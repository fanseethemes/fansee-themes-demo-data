== Installation ==

1. Upload the `fansee-themes-demo-data` folder to your `/wp-content/plugins/` directory or alternatively upload the fansee-themes-demo-data.x.x.x.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your computer.
2. Activate the Fansee Themes Demo Data WordPress plugin through the 'Plugins' menu in WordPress.
4. Goto Appearance / Import Demo Aata.
